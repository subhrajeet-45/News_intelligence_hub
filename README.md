## 📖 Overview

News Intelligence Hub is a modern AI-powered news aggregation and analysis platform built using **Google Gemini AI**. Instead of simply displaying news articles, the platform enhances the news experience through intelligent summaries, personalized recommendations, live intelligence dashboards, and AI-generated insights.

The application is designed to help users quickly understand important events, track breaking developments, and consume information more efficiently.

## ⚙️ Getting Started

### Prerequisites

Before running the project locally, ensure you have:

* Node.js (v18 or later)
* npm
* Google Gemini API Key

---

### Installation

Clone the repository:

```bash
git clone https://github.com/subhrajeet-45/News_intelligence_hub.git
cd News_intelligence_hub
```

Install dependencies:

```bash
npm install
```

---

### Environment Setup

Create a file named:

```text
.env.local
```

Add your Gemini API key:

```env
GEMINI_API_KEY=your_api_key_here
```

---

### Run the Application

Start the development server:

```bash
npm run dev
```

Open:

```text
http://localhost:5173
```
